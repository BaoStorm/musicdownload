# musicdownload

> 使用Vue开发的谷歌扩展，获取音乐下载地址
> 目前支持网易云音乐，QQ音乐，百度音乐，酷我音乐

## Build Setup

``` bash
# 安装NPM模块
npm install

# 运行 at localhost:8080
npm run dev

# 打包
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
